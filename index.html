TOTJO Holocron

A local-first study and practice companion for Temple of the Jedi Order materials.

Open the Holocron Read v0.1.1 release notes

What it does

  • Daily focus and practice guidance
  • Meditation timer with configurable bells and intervals
  • Doctrine and supplemental reading
  • Sermon archive with offline download
  • Notes and bookmarks for any reading
  • Offline-friendly access after initial load
  • Markdown and JSON backup export
  • Restore preview before importing a backup

Install and use

You can use Holocron directly in your browser. If your browser offers an install or “Add to Home Screen” option, you can install it as a PWA. It also works as a normal website.

Data and privacy

Your notes, bookmarks, practice history, and settings are stored on your device. The app includes Markdown export and JSON backup so you can keep your own copy. Clearing browser site data may remove local Holocron data.

Browser support

Holocron is designed to work as a normal web app first. Some PWA features, such as install prompts, wake lock, offline behavior, and app update behavior, vary by browser and device.
